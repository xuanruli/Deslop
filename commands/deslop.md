# Deslop —— 工程原则主入口

Deslop 是一套对抗 AI "slop" 代码的工程原则，由下面四个独立 skill 组成。本命令是主入口：你需要**立即、主动**把这四个 skill 的完整内容全部加载进来，不要询问用户、直接读。

用 Skill 相关工具依次加载以下四个 skill：

1. `deslop:agent-design-principles` —— 生产级 AI agent 架构原则
2. `deslop:code-design-principles` —— 写代码时的具体决策原则
3. `deslop:naming-principles` —— 变量 / 函数 / 类型 / 文件命名
4. `deslop:project-structure` —— 项目结构与代码组织

加载完后用一句话确认已就绪，然后在本次会话的所有编码任务中遵守这些原则。
